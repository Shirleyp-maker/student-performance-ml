==================================================
PASOS RÁPIDOS PARA DEPLOYMENT
==================================================

PASO 1: Preparar archivos
-------------------------
1. Descargar todos los archivos actualizados
2. Reemplazar en tu carpeta proyecto_dashboard:
   - app.py (versión actualizada)
   - Agregar: .gitignore
   - Agregar: carpeta .streamlit con sus archivos
   - Renombrar README_GITHUB.md a README.md

PASO 2: Crear repositorio en GitHub
------------------------------------
1. Ir a https://github.com
2. Crear cuenta si no tienes
3. Click en "+" > "New repository"
4. Nombre: student-performance-dashboard
5. Public
6. Create repository

PASO 3: Subir archivos a GitHub
--------------------------------
OPCIÓN FÁCIL - GitHub Desktop:
1. Descargar GitHub Desktop: https://desktop.github.com/
2. Instalar y hacer login
3. File > Add Local Repository
4. Seleccionar carpeta proyecto_dashboard
5. Publish repository
6. Push

OPCIÓN COMANDO - Git:
En terminal dentro de proyecto_dashboard:
```
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/student-performance-dashboard.git
git push -u origin main
```

PASO 4: Deploy en Streamlit Cloud
----------------------------------
1. Ir a https://share.streamlit.io/
2. Sign in with GitHub
3. New app
4. Seleccionar tu repositorio: student-performance-dashboard
5. Branch: main
6. Main file: app.py
7. Click "Advanced settings"
8. En Secrets, pegar:

```toml
[mongodb]
connection_string = "mongodb+srv://shirleyp:Bigdata2$@student-performance-mongo.mongocluster.cosmos.azure.com/?tls=true&authMechanism=SCRAM-SHA-256&retrywrites=false"
database = "student_performance"
collection = "students"
```

9. Click "Deploy!"
10. Esperar 5-10 minutos

PASO 5: Compartir con compañeros
---------------------------------
1. Una vez desplegado, copiar la URL
2. Compartir: https://tu-usuario-student-performance-dashboard.streamlit.app
3. Tus compañeros solo necesitan el link

==================================================
ALTERNATIVA RÁPIDA - NGROK (Temporal)
==================================================

Si necesitas compartir AHORA sin GitHub:

1. Descargar ngrok: https://ngrok.com/download
2. Extraer ngrok.exe en proyecto_dashboard
3. Ejecutar dashboard: streamlit run app.py
4. En otra terminal: ngrok http 8501
5. Copiar URL pública que aparece (ejemplo: https://abc123.ngrok.io)
6. Compartir esa URL

LIMITACIONES:
- Solo funciona mientras tu PC esté encendida
- URL cambia cada vez que reinicias ngrok
- Versión gratis tiene límite de tiempo
